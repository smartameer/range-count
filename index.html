Range Count

Introduction

Except Internet Explorer, no other browser shows the current count on sliding the slider. This plugin will allow the range slider to show current count in other browsers irresptive of it's position.

Installation

You can now install by using bower :

              > bower install range-count

(or) clone by using git :

              > git clone https://github.com/smartameer/range-count.git // https url
> git clone git@github.com:smartameer/range-count.git     // ssh url

Usage

Include Range Count library in your html code.
Note :- JQuery is a dependency. 

              <script type="text/javascript" src="jquery-range-count.min.js"></script>

Html Snippet

              <input type="range" id="myrange" max="100" min="1"/>

Jquery Snippet

              $("#myrange").rangeCount();
