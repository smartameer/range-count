Range-count

Html <input type="range"> tag's count in change of slider in non-supporting browser

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

Description

Except Internet Explorer, no other browser shows the current count on sliding the slider. This plugin will allow the range slider to show current count in other browsers irresptive of it's position.

Clone

Soon coming to bower ...

As of now you can clone the repository from https://github.com/smartameer/range-count.git and use it in your site.

Install

After cloning repository, just do a bower install in the repository as this library is dependednt on jQuery

Authors and Contributors

Add this to your html file where you are going to use the range-count 

                <script src="jquery.min.js" type="text/javascript"></script>
                <script src="range-count/dist/jquery-range-count.min.js" type="text/javascript"></script>
              
                <script>
                  $(document).ready(function() {

                     $("input[type='range']").rangeCount();

                  });
                </script>
